当凌辉 Bot 被授予群管理员或更高权限时，您理解并同意：<br>

1. 凌辉 Bot 的管理员权限仅用于其已说明或可合理预期的功能用途，不保证覆盖所有人工管理场景。

2. 由于 Bot 行为具有自动化特性，可能出现误判、延迟或执行结果与个别用户预期不一致的情况，您理解并接受该风险的存在。

3. 凌辉 Bot 的禁言、踢出、警告等管理行为，通常基于群内既定规则或管理员的配置策略自动执行。

4. 因配置不当、规则设定不清或授权不合理所产生的后果，应由相关配置或授权方自行承担，开发者不对具体管理结果承担责任。

5. 如您不再需要凌辉 Bot 执行管理功能，应及时撤销其管理员权限，而非通过异常方式干扰其正常运行。

6. 禁止通过刻意诱导、构造特殊场景、批量触发指令等方式，利用凌辉 Bot 的管理员权限制造混乱或影响群内秩序。

## 用户禁止行为

在使用凌辉 Bot 的过程中，您不得利用凌辉 Bot 从事以下行为：<br>

1. 对群内任何成员造成骚扰、误解，或泄露任何第三方的隐私信息；
2. 干扰或破坏凌辉 Bot 管理组或群管理组的正常管理与维护；
3. 在凌辉 Bot 拥有管理员权限时，恶意或无正当理由地利用凌辉 Bot 禁言、踢出群成员，或限制其使用凌辉 Bot；
4. 恶意生成大量错误、无效或冗余数据，对服务器存储或运行造成影响；
5. 针对他人数据进行恶意操作，危及他人数据安全；
6. 利用凌辉 Bot 已知或未知的漏洞（Bug）进行违规操作，或影响服务器正常运行；
7. 利用凌辉 Bot 在短时间内向第三方 API 或网站发送异常或高频请求（如使用脚本、插件或连点器恶意刷取指令）
8. 诱导或利用凌辉 Bot 生成、传播违反法律法规、暴力、低俗或他人隐私的内容；
9. 由于第三方客户端的稳定性始终差于官方客户端，所以每一次凌辉 Bot 账户被封禁，开发者都会拉取全栈日志检查是否有违规操作。若检查出以下任意行为之一，开发者有权直接封禁用户而不遵循事不过三。
    - 为了刷取游戏或虚拟资源，恶意把凌辉 Bot 拉入多个群聊。
    - 为了刷取游戏或虚拟资源，恶意向凌辉 Bot 发送多个好友请求。
    - 使用诱导、欺骗性手段绕过凌辉 Bot 现有的安全机制、利用部分功能未设限来诱导凌辉 Bot 发送过长文本（≥150 字符长度，如果系统使用了额外规则，以系统规则为准）。
    - 其他明显的违规行为。
10. 凌辉 Bot 内的所有虚拟资源（如墨辉币、道具等）仅限用于该 Bot 提供的功能体验，不具备任何法定货币属性或实物价值。开发者严禁用户利用虚拟资源进行任何形式的线下交易或非法获利。

> 日志的调取仅用于安全审计与违规判定，开发者承诺不泄露、不转让用户的私密通信内容。


不论违规对象的身份，只要您出现了本协议所列明的禁止行为，开发者将原则上遵循“事不过三”的处理方式。

但如群管理组（包括群主或群管理员）明确要求您不得继续使用凌辉 Bot，则应以群管理组的决定为准。  
在此情况下，开发者不承担任何相关责任。

若违规对象本身属于群管理组，是否限制或禁止其继续使用凌辉 Bot，应由该群自行协商决定；  
开发者仅在接到群主或群管理组的明确意见后，视情况予以配合。
